## Was ist ClassInsights?

ClassInsights ist eine Software Lösung, primär für Schulen entwickelt, mit der sich Ihre Schulcomputer vollautomatisch anhand des WebUntis-Stundenplans herunterfahren.
Weiters haben wir auch einige Monitoring Features mit denen Sie die Auslastung Ihrer Computer beobachten können.

Sollten Sie Interesse an ClassInsights haben können Sie gerne eine [kostenlose Demo anfordern](https://classinsights.at/demo)!
